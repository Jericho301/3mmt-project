 Simple Linear Regression – Marketing ROI Analysis

This project analyzes the impact of different marketing channels (TV, Radio, and Social Media) on Sales using Simple Linear Regression.

The main goal is to identify which marketing channel gives the best Return on Investment (ROI) and provide data-driven recommendations for marketing budget allocation.

 Key Objectives
- Load and clean the marketing dataset
- Perform Exploratory Data Analysis (EDA) with visualizations
- Identify the strongest predictor of Sales
- Build an OLS regression model using statsmodels
- Validate regression assumptions through diagnostic plots
- Interpret results and give business recommendations

---

Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib & Seaborn
- Statsmodels
- SciPy
- Jupyter Notebook

Analysis Summary

This project demonstrates a complete Simple Linear Regression workflow:

-Data loading and cleaning
-Exploratory Data Analysis (EDA) with pairplots, scatter plots, and correlation heatmap
-Selection of the best marketing channel based on correlation
-Fitting an Ordinary Least Squares (OLS) regression model
-Model diagnostics (Residuals vs Fitted, Q-Q Plot, Residual Distribution)
-Interpretation of R-squared, coefficients, and p-values
-Business recommendation for marketing budget allocation
